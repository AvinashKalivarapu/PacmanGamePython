Implemented using OOP principles
Simple Terminal-Based Pacman Game
With Multiple Ghosts and random generated obstacles
Represents ScoreBoard and GameBoard after each and every move

Pacman is represented by ‘P’ and Ghost by ‘G’
Coins : 'C'
Walls / Obstacles : 'X'
Moves: 
   ‘w’ : up 
   ‘s’ : down 
   ‘a’ : left 
   ‘d’ : right 
Quit ‘q’
